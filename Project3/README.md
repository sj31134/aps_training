# Project3

풀스택 크리스마스 환영 메시지 웹 애플리케이션입니다.

## 프로젝트 개요

- **목적**: 풀스택 웹 애플리케이션 개발 학습 (프론트엔드 + 백엔드)
- **기술 스택**: Node.js, Express.js, HTML, CSS, JavaScript
- **기능**: 웹페이지에서 버튼 클릭 시 백엔드에서 랜덤 크리스마스 환영 메시지를 받아 표출
- **특징**: 하나의 Node.js 환경에서 프론트엔드와 백엔드가 동시에 실행

## 파일 구조

```
Project3/
├── public/              # 프론트엔드 파일
│   ├── index.html       # 메인 HTML 파일
│   ├── styles.css       # 크리스마스 테마 스타일
│   └── app.js           # 프론트엔드 JavaScript
├── server.js            # Express 서버 (프론트엔드 + 백엔드)
├── package.json         # Node.js 의존성 관리
└── README.md            # 프로젝트 설명 (현재 파일)
```

## 사용 방법

### 1. 의존성 설치

```bash
cd Project3
npm install
```

### 2. 서버 시작

```bash
npm start
```

또는

```bash
node server.js
```

### 3. 웹페이지 접속

서버가 시작되면 브라우저에서 `http://localhost:3002` 접속

### 4. 메시지 받기

웹페이지에서 "🎁 메시지 받기 🎁" 버튼을 클릭하면 랜덤한 크리스마스 환영 메시지가 표시됩니다.

## 기능 설명

### 프론트엔드
- **크리스마스 테마 디자인**: 눈송이 애니메이션, 크리스마스 색상 테마
- **버튼 클릭 이벤트**: 클릭 시 백엔드 API 호출
- **메시지 표시**: 받은 메시지를 화면에 표출
- **로딩 상태**: API 호출 중 로딩 상태 표시
- **에러 처리**: 오류 발생 시 에러 메시지 표시

### 백엔드
- **정적 파일 서빙**: 프론트엔드 파일 제공
- **API 엔드포인트**: `/api/message` - 랜덤 크리스마스 메시지 반환
- **랜덤 메시지 생성**: 10가지 다양한 크리스마스 환영 메시지 중 랜덤 선택
- **에러 처리**: 예외 상황 처리 및 에러 응답

## API 엔드포인트

### GET /api/message

랜덤 크리스마스 환영 메시지를 반환합니다.

**요청 예시:**
```
GET http://localhost:3002/api/message
```

**응답 예시:**
```json
{
  "success": true,
  "data": {
    "message": "🎄 메리 크리스마스! 행복한 연말 되세요!",
    "emoji": "🎄",
    "timestamp": "2024-01-15T10:30:00.000Z"
  }
}
```

## 크리스마스 메시지 예시

- 🎄 메리 크리스마스! 행복한 연말 되세요!
- 🎅 산타가 당신을 찾고 있어요! 즐거운 크리스마스!
- ❄️ 눈이 내리는 크리스마스, 따뜻한 하루 되세요!
- 🎁 크리스마스 선물처럼 특별한 하루 되세요!
- 🌟 별이 반짝이는 밤, 평화로운 크리스마스 되세요!

그 외 5가지 메시지가 더 있습니다!

## 포트 변경

기본 포트는 3002입니다. 다른 포트를 사용하려면:

```bash
PORT=3003 node server.js
```

또는 PowerShell에서:

```powershell
$env:PORT=3003; node server.js
```

## 프로젝트 구조 설명

이 프로젝트는 Project1과 Project2의 구조를 결합한 풀스택 애플리케이션입니다:

- **Project1 방식**: Express 서버로 정적 파일 서빙 (프론트엔드)
- **Project2 방식**: Express 서버로 API 엔드포인트 제공 (백엔드)
- **통합**: 하나의 `server.js` 파일에서 프론트엔드와 백엔드를 모두 처리
